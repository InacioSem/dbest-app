# Changelog

## [0.1.0] - 2026-04-01

### Added
- Initial project scaffold
- Full project structure: client (Next.js App Router) + server (Express) + shared types
- 4-language i18n support: Kreyòl Ayisyen (ht), Français (fr), English (en), Español (es)
- 20 creative parameters with full translations
- 6 project types: music_video, lyric_video, visualizer, album_art, social_teaser, spotify_canvas
- Dark studio-like UI with glassmorphism design
- Demo mode for testing without API keys
- Distribution channel interface (pluggable export system)
- Style preset system (marketplace-ready)
- Landing page with upload zone, project type selector, creative params
- Processing status page with 6-step pipeline progress
- Storyboard review page with timeline
- Gallery page with downloads and export options
